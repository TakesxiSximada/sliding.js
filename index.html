sliding.js

The sliding.js is a library for displaying a list of screen capture of the web page.

There are three functions roughly to it.

DISPLAY

It lists them on the basis of the data obtained via the API.

SEARCH

You can retrieve an object based on it the title of each page are included in the data obtained via the API.

GENERATE

sliding.js offers a utilities outputter the json for the API. It will take a screen capture using the phantomjs.
